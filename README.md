# EZone - A E-Commerce Web Application
<ul>
<li> An ASP assignment, it will have basic funtionalities and minimum design. </li>
 <li> Main functionalities that will be added are: </li>
 <ol>
  <li> signup- Add new user </li>
  <li> web form </li>
